# lootboxesmult

## !!! To work properly its need a clean `LootBoxes.csv` or `LootBoxes-original.csv` file otherwise it will multiply current values !!!

# Dev
1. Install BunJS
2. Copy `LootBoxes.csv` from `[GAME_DIRECTORY]\AtomicHeart\Content\GameTune` beside `cli.ts`
3. Run `bun start`
4. Choose multiplier for loot
5. Copy new `LootBoxes.csv` to `[GAME_DIRECTORY]\AtomicHeart\Content\GameTune`
6. Start game and have fun :D

# Release
1. Download latest release for your system in Release tab
2. Put `lootboxesmult` in `[GAME_DIRECTORY]\AtomicHeart\Content\GameTune`
3. Open `lootboxesmult`
4. Choose multiplier for loot
5. Start game and have fun :D
